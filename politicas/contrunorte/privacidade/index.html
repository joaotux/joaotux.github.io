Política de Privacidade e Proteção de Dados - ContruNorte


Introdução


A ContruNorte está comprometida em proteger a privacidade e os dados pessoais dos usuários de seu aplicativo de delivery de produtos de construção. Esta Política de Privacidade descreve como coletamos, usamos e protegemos as informações pessoais dos usuários de acordo com a Lei Geral de Proteção de Dados (Lei nº 13.709/2018 - LGPD).


Coleta de Dados Pessoais


Para melhorar a experiência dos nossos clientes e oferecer serviços de qualidade, coletamos as seguintes categorias de dados pessoais:


  1. Informações de Cadastro: Nome, endereço de e-mail, número de telefone e endereço de entrega.


  2. Informações de Pagamento: Dados do cartão de crédito para processamento de pagamentos.


  3. Informações de Uso: Informações sobre como você utiliza o aplicativo ContruNorte, como histórico de pedidos e preferências de compra.


Finalidades do Tratamento de Dados


Os dados pessoais coletados são utilizados para as seguintes finalidades:


  1. Processar pedidos de produtos de construção.

  2. Realizar a entrega dos produtos no endereço cadastrado.


  3. Processar pagamentos de acordo com as transações realizadas.


  4. Melhorar e personalizar a experiência do usuário no aplicativo.


Compartilhamento de Dados


A ContruNorte não compartilha seus dados pessoais com terceiros, exceto quando necessário para cumprir obrigações contratuais,

legais ou regulatórias.


Segurança dos Dados


Implementamos medidas de segurança adequadas para proteger seus dados pessoais contra acesso não autorizado, alteração, divulgação ou destruição. Utilizamos criptografia para proteger as informações de pagamento.


Retenção de Dados


Manteremos seus dados pessoais pelo tempo necessário para cumprir as finalidades mencionadas nesta política, a menos que uma retenção mais longa seja exigida ou permitida por lei.


Direitos dos Titulares dos Dados


De acordo com a LGPD, você tem o direito de:

  • Acessar seus dados pessoais.


  • Corrigir seus dados pessoais, caso estejam incorretos ou desatualizados.


  • Excluir seus dados pessoais, exceto quando a lei exige a retenção dos mesmos.


  • Revogar o consentimento para o tratamento dos dados pessoais, quando o consentimento for a base legal para o tratamento.


  • Obter informações sobre o compartilhamento dos seus dados pessoais.

Alterações na Política de Privacidade


Esta Política de Privacidade pode ser atualizada periodicamente para refletir mudanças em nossas práticas de privacidade.

Recomendamos que você reveja esta política periodicamente.


Ao utilizar o aplicativo ContruNorte, você concorda com os termos desta Política de Privacidade e com o tratamento dos seus dados pessoais de acordo com as finalidades aqui descritas.
