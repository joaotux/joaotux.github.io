Principais Projetos

ERP Edukaa


Sistema desktop desenvolvido para a escola Integração de São Bernado do Campo - SP.

O sistema permite o controle das matrículas dos alunos, lançamento de despesas dos alunos, módulos de receber e a receber, controle dos funcionário, controle de folha de pagamentos, conciliação bancária e módulo de relatórios.

O sistema foi desenvolvido com o objetivo de ser rápido e fácil de ser operado pelos usuários. Ele é multi-plataformas, podendo ser usado em ambientes Windows e Linux.

Tecnologias usadas: Spring, JavaFX, MariaDB

PDV Imagens


Sistema desktop desenvolvido para a empresa Noel Mania de Santa Catarina, que organiza e vende material fotográfico em espaços decorados e destinados à datas comemorativas.

O sistema permite a venda de imagens impressas e digitais, ele realiza a contabilização financeira de todas as imagens impressas ou enviadas por e-mais para os clientes de forma automática.

O sistema possui os seguintes módulos, controle de usuários, caixa, pedidos, e-mails enviados, cadastro de produtos, controle de estoque, cadastro de impressoras padrões, auditoria e relatórios.

Tecnologias usadas: Spring, JavaFX, MariaDB

Gestor de Traduções


Sistema web desenvolvido para realizar o gerenciamento de projeto de traduções de agencias da região do Rio de Janeiro, como os de artigos, vídeos e etc.
O sistema permite o controle total de todas as rotinas de uma agência de tradução, permitindo o gerenciamento dos projetos em andamento, gerenciamento dos clientes, geração de relatórios, geração de orçamentos, controle de pagamento dos funcionários e terceirizados, glossário em português, inglês e espanhol para auxiliar nas traduções, agenda de contatos e lembrete de aniversários de clientes.

O sistema possui internacionalização para português, inglês e espanhol.

Tecnologias usadas: Spring, Java, MySQL

Delivery ConstruNorte

O sistema possui uma retaguarda web através da qual a empresa pode gerenciar os pedidos recebidos e possui um sistema desktop que permite a integração do estoque dos produtos com o sistema de ERP da empresa.
